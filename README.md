# fi-tracker

Familieidræt tracker: fundraising, fund pipeline, tasks, ideas, and årshjul.

## Setup

```bash
pip install -r requirements.txt
python3 app.py
```

Open http://localhost:5002

## Modules

| Module | Description |
|---|---|
| Dashboard | Overview stats |
| Fundraising | Grants/donations received or applied for |
| Fund Pipeline | Funds to research and apply for |
| Tasks | Tasks with assignee, priority, due date |
| Ideas | Idea board with category + status |
| Årshjul | Annual event calendar (placeholder) |
